# 影视匠 OBS 管理工具箱

> 专为 OBS Studio 打造的专业管理工具，零依赖，开箱即用

## ✨ 功能特性

- 🔍 **进程扫描** — 实时扫描 OBS 相关进程（OBS / ASUS / 推流 / 其他）
- 🔧 **一键修复** — 诊断 + 修复 OBS 常见问题（卡顿/崩溃/黑屏）
- 🧹 **插件管理** — 扫描、排序、卸载 OBS 插件（含关联文件）
- 💾 **配置备份** — 备份和重置 OBS 配置文件
- 📊 **系统信息** — 显示 OBS 安装路径、插件数量统计
- 🌙 **深色主题** — 精心调校的专业暗色界面

## 📦 下载安装

前往 [Releases](../../releases) 下载最新版 `影视匠OBS管理工具箱.exe`，双击运行，无需安装任何依赖。

## 🖥️ 系统要求

- Windows 10 / 11 (64位)
- OBS Studio（已安装）
